## Pages (7 React pages)

1. **Home** — Hero video/image, quick stats, featured classes, trainer spotlight, testimonials, pricing preview, CTA
2. **Classes** — Filterable class schedule (by day, type, trainer), class cards with time/duration/intensity
3. **Trainers** — Trainer grid with specialties, certifications, class schedule, individual trainer profiles
4. **Pricing** — 4-tier pricing with comparison, membership benefits, FAQ
5. **About** — Studio story, mission, timeline, community
6. **Contact** — Form, studio location, hours, social links
7. **Nutrition** — Meal plans, nutrition coaching info, success stories
