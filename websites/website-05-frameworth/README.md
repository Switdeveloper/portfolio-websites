# FrameWorth — Photography Portfolio

**Niche:** Photography Portfolio / Creative Agency
**Vibe:** Elegant, minimal, editorial — letting photos breathe. Luxury boutique feel.
**Color Palette:**
- Primary: `#1A1A1A` (near black)
- Background: `#FAFAFA` (warm off-white)
- Accent: `#C9A84C` (warm gold)
- Text: `#1A1A1A`
- Text muted: `#888888`
- Light: `#F0F0F0`
- Border: `#E0E0E0`

## Pages (6 React pages)

1. **Home** — Fullscreen hero with featured image, intro statement, featured work grid, services overview, client logos, testimonials, CTA
2. **Portfolio** — Masonry-style gallery with category filter (All, Portrait, Wedding, Commercial, Landscape, Street), lightbox modal
3. **Services** — Service packages (Photography, Videography, Editing), pricing, process steps
4. **About** — Photographer story, equipment, awards/press, timeline
5. **Journal** — Blog-style articles (mocked), categories, reading time
6. **Contact** — Inquiry form, booking calendar placeholder, location info, social links

## Functional Features

- ✅ Portfolio gallery with category filtering
- ✅ Lightbox/modal for full-size image viewing (click to open, arrow nav, close)
- ✅ Masonry-style layout for portfolio grid
- ✅ Smooth scroll navigation
- ✅ Scroll-triggered animations (fade-in on scroll)
- ✅ Sticky header with shadow on scroll
- ✅ Back-to-top button
- ✅ Testimonials section
- ✅ Contact form with validation
- ✅ Mobile hamburger menu
- ✅ Active nav link highlight
