# Verdant & Co — Premium Plant Shop

**Niche:** Eco-friendly plant e-commerce
**Vibe:** Organic, warm, premium, earthy
**Color Palette:**
- Primary: `#2D5A27` (deep forest green)
- Secondary: `#8B6914` (warm amber)
- Accent: `#E8F5E3` (soft mint)
- Background: `#FAFAF7` (warm off-white)
- Text: `#1A2E1A` (dark forest)

## Pages

1. **Home** — Hero with tagline, featured plants, value props, newsletter
2. **Shop** — Product grid with category filters, search, add-to-cart
3. **Plant Care** — Care guides by plant type, watering tips, light requirements
4. **About** — Brand story, sustainability mission, team
5. **Contact** — Contact form, store hours, location map
6. **Cart** — Full cart management, quantity update, checkout button

## Functional Features

- ✅ Add to cart (with quantity)
- ✅ Remove from cart
- ✅ Update quantity
- ✅ Cart persists in localStorage
- ✅ Category filtering (Indoor, Outdoor, Succulents, Rare)
- ✅ Search plants by name
- ✅ Newsletter signup form
- ✅ Contact form with validation
- ✅ Responsive mobile navigation

## Plant Data (12 plants)

Realistic plant inventory with names, prices, categories, care difficulty, light/water needs

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- React Router DOM
- Context API for cart state
